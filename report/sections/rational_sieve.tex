        L'algorithme n'utilise que les variables $A$ et $b$ (on considerera les variables $i$, $j$ et $k$ commes negligeables).\\
        La complexite en espace de l'algorithme est donc : $$\lvert A \rvert + \lvert b \rvert = n \cdot n + n = n ^2 +n$$
